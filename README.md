## 📖 Overview

Cloud Resume Builder is a modern web application designed to simplify resume creation for students, freshers, and professionals.

Instead of struggling with formatting in Word or Google Docs, users can simply fill out their information through an intuitive interface. The application automatically generates a clean, professional, ATS-friendly resume with real-time preview and PDF export support.

Whether you're applying for internships, jobs, or freelance opportunities, Cloud Resume Builder helps you create a polished resume within minutes.

# 🚀 Getting Started

## 1. Clone the repository

```bash
git clone https://github.com/yourusername/cloud-resume-builder.git
```

---

## 2. Navigate to project folder

```bash
cd cloud-resume-builder
```

---

## 3. Install dependencies

```bash
npm install
```

---

## 4. Start development server

```bash
npm run dev
```

---

## 5. Build for production

```bash
npm run build
```

---

## 6. Preview Production Build

```bash
npm run preview
```
